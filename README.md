# QREncoderGUI
QRCode encoder with GUI based on [ZXing.NET](http://zxingnet.codeplex.com/)

## Usage

You need to add [ZXing.NET](http://zxingnet.codeplex.com/) reference in Visual Studio before running `QREncoderGUI`.
